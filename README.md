# Bellabeat Smart Device Usage Analysis

## Project Overview
This project analyzes smart device usage data to identify trends in user activity, sedentary behavior, and wellness tracking habits.

The goal is to generate insights that could help guide Bellabeat’s marketing strategy.

## Tools Used
- Excel
- Pivot Tables
- Data Visualization
- GitHub

## Dataset
FitBit Fitness Tracker Dataset (Kaggle)

## Key Analysis Areas
- Average daily activity by user
- Monthly activity trends
- Activity intensity distribution
- Sleep duration analysis
- Relationship between activity and calories burned
- Activity patterns by day of week

## Key Insights
- Users show large variation in activity levels
- Sedentary behavior represents a large portion of daily activity
- Activity levels fluctuate across days of the week
- Higher activity levels generally correspond to higher calorie expenditure
